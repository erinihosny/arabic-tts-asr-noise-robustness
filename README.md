# Arabic Dialect ASR Robustness Evaluation

This repository contains code for evaluating the robustness of Arabic ASR
across multiple dialects using TTS-generated speech under controlled noise conditions.

## Dialects
- MSA
- Egyptian
- Saudi
- Levantine
- Moroccan

## Models
- XTTS v2 (TTS)
- Whisper Base (ASR)

## Evaluation
